SynChrono module

In many autonomous vehicle simulations, you don't care what happens if two cars collide - that would be bad so you just end the simulation. In SynChrono we take advantage of that fact and split a simulation of many Chrono::Vehicles across multiple MPI ranks. This let us scale from running ~1 vehicles in real-time to running ~100 vehicles in real-time. The code is in the Chrono repository, and a longer description is on our lab website.
